# DNA Playground

## Usage
* `make build`
* `make start`
* `make stop`

## Resources
* Biopython: [website](https://biopython.org), [tutorial](http://biopython.org/DIST/docs/tutorial/Tutorial.html)
* Nextstrain: [website](https://nextstrain.org), [github](https://github.com/nextstrain)
